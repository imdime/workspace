# Video/Gif script
#
####Usage (need jQuery to run)

html :
```html
<div class="gif-wrapper" data-video-url="https://www.youtube.com/embed/VYOjWnS4cMY" data-gif-url="https://juppijuppsen.com/wp-content/uploads/2017/01/Nature.gif"></div>
```          
js :
```javascript
$('.gif-wrapper').initVideo();
```
